# 教师工具箱 (Teacher Toolbox)

一个功能全面的中学教师在线工具箱网站，集成多种实用教学工具，支持多用户访问和模块化扩展。

## 🚀 功能特色

### 基础工具模块
- **自习课分贝检测**：实时监测课堂噪音水平，提供可视化反馈
- **异常记录管理**：记录大屏/全屏使用时间，生成使用报告
- **多功能计算器**：支持科学计算、单位换算等教学常用计算
- **3D地球仪**：展示最新地理数据，支持交互式教学演示

### 课堂管理工具
- **随机点名系统**：支持多种点名模式（单人、分组、排除名单）
- **课堂计时器**：倒计时、正计时、分段计时功能
- **分组工具**：自动分组、手动调整、分组结果保存

### 教学辅助工具
- **成绩统计分析**：成绩录入、统计图表、成绩分布分析
- **教案模板库**：多学科教案模板、自定义模板编辑
- **多媒体资源管理**：课件、视频、图片资源分类管理

### 学科专用工具
- **数学公式编辑器**：LaTeX公式编辑、图形绘制
- **化学元素表**：元素周期表、化合物查询、反应模拟
- **历史时间轴**：历史事件时间线、关联知识点

### 互动协作工具
- **在线白板**：多人协作绘图、板书共享
- **投票工具**：快速投票、结果统计分析
- **知识问答系统**：题库管理、自动组卷、答题统计

## 🛠️ 技术栈

- **前端**：React + TypeScript + Tailwind CSS + Three.js
- **后端**：Node.js + Express.js + TypeScript
- **数据库**：PostgreSQL + Redis
- **实时通信**：Socket.IO + WebRTC
- **部署**：Docker + Nginx

## 📁 项目结构

```
教师工具箱/
├── frontend/          # 前端应用
├── backend/           # 后端API服务
├── shared/            # 共享类型定义
└── docker-compose.yml # 容器编排配置
```

## 🚀 快速开始

### 环境要求
- Node.js 18+
- PostgreSQL 14+
- Redis 6+

### 安装依赖
```bash
npm run install:all
```

### 开发模式
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📦 部署

使用 Docker 快速部署：
```bash
docker-compose up -d
```
